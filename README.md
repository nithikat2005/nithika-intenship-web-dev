
# Portfolio Website - Nithika M

This is a simple and responsive personal portfolio website built using HTML and CSS. It showcases services, skills, and contact details in a clean layout.

## Features
- Responsive design for all devices  
- Easy navigation (Home, About, Services, Contact)  
- Clean and modern UI  

## Files
- index.html – Home page  
- about.html – About section  
- services.html – Services offered  
- contact.html – Contact details  
- style.css – Main stylesheet
